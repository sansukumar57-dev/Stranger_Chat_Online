# 💬 Stranger Chat Online

A real-time anonymous chat application that instantly connects users with random strangers for one-on-one conversations. No account required—just join and start chatting!

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Socket.IO](https://img.shields.io/badge/Socket.IO-RealTime-black)
![Node.js](https://img.shields.io/badge/Node.js-Backend-green)

---

## 🚀 Features

- 🔀 Random stranger matching
- 💬 Real-time messaging using WebSockets
- ⚡ Instant message delivery
- 🟢 Online user tracking
- 🔒 Anonymous chatting
- 📱 Responsive UI
- 🚪 Leave chat and find a new stranger
- 🎨 Clean and modern interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/sansukumar57-dev/Stranger_Chat_Online.git
```

Navigate into the project

```bash
cd Stranger_Chat_Online
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```

or

```bash
node server.js
```

Open your browser

```
http://strangers-chat-online.vercel.app/
```

## 🔄 How It Works

1. User opens the application.
2. Server searches for another available user.
3. Two users are paired instantly.
4. Messages are exchanged in real time using Socket.IO.
5. Either user can leave and be matched again.
